# Reflection

This assignment is my first experience building a full-stack application using Node.js and TypeScript.

## Key Learnings

* Understood how backend servers work using Express
* Learned how APIs are created and how request-response cycle works
* Gained familiarity with TypeScript configuration
* Learned the importance of debugging and environment setup

## AI Usage

AI helped significantly in:

* Setting up the backend quickly
* Debugging errors efficiently
* Explaining concepts in a simple way

However, I also learned that:

* AI output needs to be verified
* Understanding concepts requires asking follow-up questions

## Challenges Faced

* Setting up Git and npm environment
* Fixing TypeScript configuration issues
* Understanding backend concepts like middleware and APIs

## Improvements for Future

* Plan architecture more clearly before coding
* Add database earlier instead of using in-memory data
* Improve frontend integration skills

## Conclusion

This assignment helped me understand backend development from scratch and how to effectively use AI tools for learning and implementation.
